
# 💕 Valentine's Website for Nenye

## Overview
A romantic, mobile-first single-page website with soft pink gradients, elegant typography, interactive buttons, and playful animations — all designed to feel personal and intentional.

## Design & Theme
- **Colors**: Soft pink gradients (#FFB6C1, #FF69B4, #FFC0CB) with white background sections
- **Typography**: Elegant script font (e.g., "Great Vibes" or "Dancing Script") for headings, clean sans-serif (e.g., "Inter") for body
- **Layout**: Centered, minimal, mobile-first with generous whitespace
- **Animations**: Slow-floating hearts in the background (subtle, classy CSS animations), smooth fade-in transitions as sections appear

## Page Sections (Top to Bottom)

### 1. Hero — "Hi Nenye 💕"
- Large elegant script heading with a soft pink gradient background
- Subtle floating heart particles drifting slowly in the background

### 2. Love Letter Message
- The personal message displayed in clean typography on a white section
- Mentions spaghetti 🍝💖 with warmth and sincerity
- Hidden spaghetti emoji elements subtly placed in the background design

### 3. The Big Question — "Will you be my Valentine? 💌"
- Two buttons: **Yes 💕** and **No 😏**
- **Yes**: Triggers a celebration screen — hearts burst animation, soft glow, and the message: *"Yessss Nenye!!! 💖 You just made me the happiest guy alive."*
- **No (1st click)**: Playful popup: *"Just dey play!!!!!, okay now, your phone will self-destruct in 30 seconds if you don't go back and click yes 😭💣"*
- **No (2nd click)**: Popup: *"It's like you don't like this your xr 😭"*
- **After 2nd No**: The "No" button starts dodging — moves to random positions when hovered or tapped

### 4. Footer
- *"Made with love, always."* in elegant script

### 5. Background Music Toggle
- Small music note icon (muted by default) in the corner to toggle a romantic instrumental track

## Technical Approach
- Single-page React app, no backend needed
- CSS animations for floating hearts and transitions
- Google Fonts for elegant typography
- Fully responsive, optimized for mobile viewing
